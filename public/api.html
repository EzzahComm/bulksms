SMS API

Send SMS from your own app with a single authenticated POST request. Free access on every plan.

One endpoint. That's it.

Generate an API key from your dashboard, then send. We validate numbers, count GSM segments, charge credits and return a per-recipient result.

  • Simple REST + JSON
  • API key authentication
  • Single or bulk recipients
  • Delivery reports via webhook
Get your API key
POST /api/sms/send
curl -X POST https://sms.ezzahcomm.co.ke/api/sms/send \
  -H "Authorization: ApiKey ezk_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello from EZZAHCOMM",
    "to": ["0712345678", "0790000000"],
    "sender": "EZZAH"
  }'
Response
{
  "campaign_id": "…",
  "status": "sent",
  "recipients": 2, "valid": 2, "invalid": [],
  "segments": 1, "credits_charged": 2,
  "sent": 2, "failed": 0
}

Response codes

CodeMeaning
200Successful request
1001Invalid Sender ID
1003Invalid mobile number
1004Low bulk credits
1006Invalid credentials (API key or partner ID)
